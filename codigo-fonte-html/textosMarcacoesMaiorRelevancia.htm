Esta página apresenta trechos de códigos do algoritmo de busca binária.

O nome da busca vem do inglês binary search algorithm. 

        public static int buscaBinaria(int[] array, int valor){
            int inicio = 0;
            int fim = array.length-1;
            
            while(inicio <= fim){
                int meio = (inicio+fim)/2;
                
                if(array[meio] == valor){
                    return meio;
                }
                if(valor > array[meio]){
                    inicio = meio+1;
                } else {
                    fim = meio-1;
                }
            }
            return -1;
        }

O algoritmo inicia criando as variáveis inicio e fim. O programa iniciará quando o usuário pressionar a tecla ENTER, e apresentará o resultado no final, que pode ser 5 BUSCAS.
